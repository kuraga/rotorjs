# RotorJS Application Example

## About

This is a [RotorJS](https://github.com/kuraga/rotorjs) application example.

It uses a default RotorJS middleware and is ECMAScript2019- and ESM-compatible.

## Usage

```sh
npm install
npm run build
open dist/index.html
```
